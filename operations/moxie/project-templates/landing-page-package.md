# Moxie project template — Landing page package

**Template name in Moxie:** `DG — Landing page (design + build)`

**Product SKU:** DG-LP-FIXED

**Fee type:** Fixed

**Source:** Extracted from Talk With Marie / Top Realty delivery pattern.

**Naming when live:** `[Client] — Landing page — [Year]`

---

## Tasks (in order)

| # | Task | Typical offset | Description |
|---|------|----------------|-------------|
| 1 | Intake / kickoff | Day 0 | Goals, audience, host, assets |
| 2 | Develop design brief & mockup | Day 1–5 | Summary of request + proposed layout |
| 3 | Client review of brief & mockup | Day 5–7 | Approval or revision notes |
| 4 | Build website | Day 7–14 | Implement approved design on client host |
| 5 | Review & revisions | Day 14–18 | Client review of build |
| 6 | Create stylesheet (optional) | Day 14–16 | Brand guide for future updates |
| 7 | Website handoff | Day 18–21 | Final QA, maintenance walkthrough, close email |

## Sub-tasks

**Task 2:** Gather copy and images from client

**Task 5:** Track revision rounds (cap at 2 in contract)
