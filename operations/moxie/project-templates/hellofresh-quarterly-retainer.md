# Moxie project template — HelloFresh quarterly retainer

**Template name in Moxie:** `DG — HelloFresh quarterly retainer`

**Product SKU:** DG-UX-HR (hourly, $135/hr default)

**Fee type:** Hourly — bill from time tracker to invoice

**Client:** Grocery Delivery E-Services USA Inc. (HelloFresh US)

**Naming when live:** `HelloFresh US — Q[N] [Year]` (example: `HelloFresh US — Q2 2026`)

---

## Project settings

| Setting | Value |
|---------|--------|
| Start | First day of quarter |
| Due | Last day of quarter |
| Invoice prefix | HF- |
| Link invoices | This project only for the quarter |

## Tasks (create each quarter)

| Task | Purpose |
|------|---------|
| `[YYYY]-Q[N] — DIY program studies` | DIY studies, Sprig, office hours tied to DIY program |
| `[YYYY]-Q[N] — Ad hoc UX research` | VOC reports, UX library, ad hoc meetings |
| `[YYYY]-Q[N] — Competitor tracker` | Competitor tracker deliverables |

Add **`[YYYY]-Q[N] — Deep dives`** only if that workstream is in the contract for the quarter.

## End of quarter

- [ ] Mark project **Complete**
- [ ] Confirm all time **Billed** or invoiced
- [ ] Spawn next quarter from this template

## Do not

- Create separate top-level projects per study type (old DIY Program / UX Research Ad Hoc pattern)
- Log generic meetings as Opportunities — use calendar + time on appropriate task
