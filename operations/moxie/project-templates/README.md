# Moxie project templates — Design Grove

Create each file below in Moxie under **Projects → Templates → + Add template**. Copy task names, order, and fee settings from the linked doc.

| Template file | Moxie template name | Fee type | SKU |
|---------------|---------------------|----------|-----|
| [nonprofit-service-mapping.md](nonprofit-service-mapping.md) | DG — Service mapping (standard) | Fixed | DG-MAP-STD |
| [landing-page-package.md](landing-page-package.md) | DG — Landing page (design + build) | Fixed | DG-LP-FIXED |
| [hellofresh-quarterly-retainer.md](hellofresh-quarterly-retainer.md) | DG — HelloFresh quarterly retainer | Hourly | DG-UX-HR |
| [merch-strategy-sprint.md](merch-strategy-sprint.md) | DG — Merch strategy sprint *(optional)* | Fixed | DG-MERCH-SPRINT |

After creating templates, attach them to **agreement templates** and enable **signed agreement → create project**.
